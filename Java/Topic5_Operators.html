// Integer division — decimal part is dropped!
int a = 10 / 3;       // 3 (not 3.33)
double b = 10.0 / 3; // 3.3333 (use double for decimal result)
double c = (double) 10 / 3; // 3.3333 (cast first)

boolean isEven = (10 % 2 == 0); // true
boolean isOdd  = (7  % 2 != 0); // true

String s = "Age: " + 25;   // "Age: 25"
System.out.println(1 + 2 + "3"); // "33" (1+2=3, then 3+"3"="33")
System.out.println("1" + 2 + 3); // "123" (left to right)
⚠️ Famous interview trick — + with Strings:
System.out.println(1 + 2 + "3") → "33" (1+2=3, then "3"+"3"="33")
System.out.println("1" + 2 + 3) → "123" (left to right, first "1"+2="12", then "12"+3="123")
Evaluated left to right — once a String is encountered, rest is concatenation!

Logical Operators

Used to combine multiple boolean conditions.

OperatorNameRuleExample
&&Logical ANDtrue only if BOTH are truetrue && false → false
||Logical ORtrue if AT LEAST ONE is truetrue || false → true
!Logical NOTreverses the boolean!true → false

Short-Circuit Evaluation — Very Important!

&& Short-circuit: If the LEFT side is false, the RIGHT side is NOT evaluated — result is already false.
|| Short-circuit: If the LEFT side is true, the RIGHT side is NOT evaluated — result is already true. 
int x = 5;

// && short-circuit — second condition not checked if first is false
if (x > 10 && (x++ > 0)) { }   // x++ never runs!
System.out.println(x);  // still 5

// || short-circuit — second condition not checked if first is true
if (x > 0 || (x++ > 0)) { }    // x++ never runs!
System.out.println(x);  // still 5

// Practical use — null check before method call
if (str != null && str.equals("hello")) { }
// Safe! If str is null, .equals() is never called → no NullPointerException
💡 & vs && and | vs ||:
&& and || — short-circuit (lazy) — skip right side if result is determined
& and | — non-short-circuit (eager) — always evaluate both sides
Always prefer && and || in conditions!

Unary Operators — Pre/Post Increment
OperatorNameExampleMeaning
++Pre-increment++xIncrement FIRST, then use value
++Post-incrementx++Use value FIRST, then increment
--Pre-decrement--xDecrement FIRST, then use value
--Post-decrementx--Use value FIRST, then decrement
-Negation-xNegates the value
!NOT!trueReverses boolean
int a = 5;
System.out.println(a++);  // 5 — prints THEN increments (a becomes 6)
System.out.println(a);    // 6

int b = 5;
System.out.println(++b);  // 6 — increments THEN prints
System.out.println(b);    // 6

// Tricky combined example
int x = 5;
int y = x++ + ++x;
// Step 1: x++ → uses 5, then x becomes 6
// Step 2: ++x → x becomes 7, uses 7
// y = 5 + 7 = 12, x = 7
System.out.println(y);  // 12
System.out.println(x);  // 7
⚠️ Memory trick:
Pre (++x) → Prepare first (increment), then use
Post (x++) → Postpone the change (use first, increment later)

Ternary Operator

The ternary operator is a shorthand for if-else. It is the only operator that takes 3 operands.

Syntax: condition ? valueIfTrue : valueIfFalse 
// if-else version
int a = 10, b = 20;
int max;
if (a > b) max = a;
else max = b;

// Ternary version — cleaner!
int max2 = (a > b) ? a : b;   // 20

// Nested ternary — avoid if too complex!
int score = 75;
String grade = (score >= 90) ? "A"
              : (score >= 70) ? "B"
              : (score >= 50) ? "C"
              : "F";
System.out.println(grade);  // B

// Practical use — null check
String name = (user != null) ? user.getName() : "Guest";

Bitwise Operators

Operate on individual bits of integer values. Less common in interviews but important to know.

OperatorNameExample (5=0101, 3=0011)Result
&Bitwise AND5 & 3 → 0101 & 00110001 = 1
|Bitwise OR5 | 3 → 0101 | 00110111 = 7
^Bitwise XOR5 ^ 3 → 0101 ^ 00110110 = 6
~Bitwise NOT~5 → ~0101-6
<<Left shift5 << 110 (multiply by 2)
>>Right shift5 >> 12 (divide by 2)
💡 Practical uses of bitwise:
n & 1 → check if n is odd (result 1) or even (result 0)
n << 1 → multiply by 2 (fast!)
n >> 1 → divide by 2 (fast!)
a ^ b ^ a → gives back b (used to swap without temp variable)

instanceof Operator

Checks if an object is an instance of a specific class or interface. Returns boolean.

String s = "Hello";
System.out.println(s instanceof String);   // true
System.out.println(s instanceof Object);   // true (String extends Object)

// Safe casting pattern
Object obj = "Hello";
if (obj instanceof String) {
    String str = (String) obj;  // safe cast
    System.out.println(str.length());
}

// Java 16+ pattern matching for instanceof
if (obj instanceof String str) {  // declares 'str' directly!
    System.out.println(str.length());
}

// null check
System.out.println(null instanceof String);  // always false

int result2 = (2 + 3) * 4;  // 20 (parentheses first)

boolean b = 5 > 3 && 2 < 4 || 1 == 2;

// Step 2: true && true = true
// Step 3: true || false = true
System.out.println(b);  // true

Top Interview Questions & Model Answers
Q1. What is the output? System.out.println(1 + 2 + "3"); and System.out.println("1" + 2 + 3);
First prints "33" — evaluated left to right: 1+2=3, then 3+"3"="33".
Second prints "123" — "1"+2="12", then "12"+3="123". Once a String appears, + becomes concatenation.
Q2. What is the difference between i++ and ++i?
i++ (post-increment) uses the current value first, then increments. ++i (pre-increment) increments first, then uses the value. Example: if i=5, x = i++ gives x=5, i=6. x = ++i gives x=6, i=6.
Q3. What is short-circuit evaluation in Java?
With &&, if the left condition is false, the right side is not evaluated (result is already false). With ||, if the left condition is true, the right side is not evaluated. This is useful for null checks: if (obj != null && obj.method()) — if obj is null, the method is never called, preventing NullPointerException.
Q4. What does the % operator do? How do you check if a number is even?
The % (modulus) operator returns the remainder of division. To check if a number is even: n % 2 == 0. If remainder is 0, the number is even. If remainder is 1, it is odd.
Q5. What is the output? int x = 5; int y = x++ + ++x;
y = 12, x = 7. Step 1: x++ uses 5 then x becomes 6. Step 2: ++x makes x=7 then uses 7. So y = 5 + 7 = 12 and x = 7.
Q6. What is the ternary operator? Give an example.
The ternary operator is a shorthand for if-else with syntax: condition ? valueIfTrue : valueIfFalse. It is the only operator with 3 operands. Example: int max = (a > b) ? a : b; — returns a if a is greater, otherwise b.
Q7. What is the difference between & and && in Java?
&& is the logical AND with short-circuit — if the left side is false, right side is skipped. & is the bitwise AND (also works as logical AND without short-circuit) — always evaluates both sides. Always prefer && in conditions for safety and performance.
